# Operating System

1. 基本特征
	* 并发
	* 共享
	* 虚拟
	* 异步
* 系统调用
	* 如果一个进程在用户态需要使用内核态的功能，就进行系统调用从而陷入内核，有操作系统代为完成
	* 进程控制 fork(),exit(),wait()
	* 进程通信 pipe()
	* 文件操作
	* 设备操作
	* 信息维护
	* 安全
* 进程管理
	* 进程是资源分配的基本单位
	* 线程是独立调度的基本单位，一个进程的多个线程共享进程资源
	* 系统开销：创建或销毁进程时，系统都要为之分配或回收资源，所付出的开销远大于创建或撤销线程的开销。
